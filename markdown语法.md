# 二. 常用快捷键

1. 代码块：Ctrl+Shift+K
2. 后退一步（撤销）：Ctrl+z
3. 前进一步：Ctrl+Y
4. 引用：Ctrl+Shift+Q
5. 几级标题：Ctrl+数字
6. 提高标题级别：Ctrl+=
7. 减低标题级别：Ctrl+-
